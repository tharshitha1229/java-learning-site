Java Programming for Beginners

A quick guide to getting started with Java

What is Java?

Java is a high-level, class-based, object-oriented programming language designed to let programmers write once, run anywhere (WORA).

Your First Program

Every Java application begins with a class name, and the program is executed from the main method.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
